TOTORIAL INSTAL WINDOWS

Berikut adalah langkah-langkah untuk menginstal Windows 10 menggunakan flashdisk: Unduh File ISO Windows 10: Kunjungi situs resmi Microsoft dan unduh file ISO Windows 10 menggunakan Media Creation Tool1. Buat Bootable USB: Gunakan software seperti Rufus untuk membuat bootable USB dari file ISO yang telah diunduh2. Pilih file ISO dan pastikan pengaturan partisi sesuai dengan tipe hard disk (GPT atau MBR). Atur Boot Priority di BIOS: Colokkan USB ke komputer dan restart. Masuk ke BIOS dengan menekan tombol seperti Del, F2, atau Esc saat komputer mulai menyala. Atur boot priority agar USB menjadi prioritas pertama. Mulai Instalasi Windows: Simpan pengaturan BIOS dan restart komputer. Komputer akan boot dari USB dan memulai proses instalasi Windows. Pilih bahasa, waktu, dan metode input, lalu klik “Install Now”. Pilih Edisi dan Lisensi: Masukkan product key jika diminta, atau pilih untuk memasukkan nanti. Pilih edisi Windows yang ingin diinstal. Partisi Hard Disk: Pilih “Custom: Install Windows only (advanced)”. Hapus partisi lama jika perlu, atau buat partisi baru sesuai kebutuhan. Pilih partisi tempat Windows akan diinstal dan klik “Next”. Tunggu Proses Instalasi: Proses instalasi akan memakan waktu beberapa menit. Komputer akan restart beberapa kali selama proses ini. Konfigurasi Awal: Setelah instalasi selesai, ikuti langkah-langkah konfigurasi awal seperti memilih wilayah, layout keyboard, dan menghubungkan ke jaringan internet. Buat akun pengguna dan atur preferensi lainnya. Instal Driver dan Aplikasi: Setelah masuk ke desktop, instal driver yang diperlukan dan aplikasi yang kamu butuhkan.
